# Scalable Ecommerce Platform

A scalable e-commerce platform built with Python, showcasing backend architecture, data handling, and business logic for a real-world online store.

---

## 🚀 Features
- Product management (add, remove, update products)
- Shopping cart logic
- Order processing
- Data persistence (file or database)
- Input validation and error handling

---

## 🛠 Tech Stack
- Python

## ▶️ How to Run

1.	Clone the repository:
  git clone https://github.com/Andrii-devPython/scalable-ecommerce-platform.git

2.	Navigate to the project folder:
  cd scalable-ecommerce-platform

3.	Run the program:
  python3 project.py

## 🧪 Running Tests

pytest
